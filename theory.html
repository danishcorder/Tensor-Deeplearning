What is a Tensor?

A tensor is a multi-dimensional array â€” a generalization of scalars (0D), vectors (1D), and matrices (2D) to higher dimensions (3D, 4D, etc.).

  • A grayscale image is a 2D tensor (Height x Width).
  • An RGB image is a 3D tensor (Height x Width x Channels).
  • A batch of RGB images is a 4D tensor (Batch x Height x Width x Channels).

Tensors allow deep learning frameworks to generalize data representations and computations for images, text, audio, and more.
